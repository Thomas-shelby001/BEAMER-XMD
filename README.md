## 🎯 About BEAMER XMD

**BEAMER XMD** is a car‑enthusiast web platform focused on **BMW** (Beamer) culture.  
It serves as a gallery, build blog, and community hub where owners and fans can:

- 📸 Browse stunning BMW builds (E30, E46, G80, etc.)
- 🔧 Share modification stories and part lists
- 🏁 Get inspired for your next project
- 🔗 Generate WhatsApp bot session IDs (integrated tool)

> ⚡ **Live Demo:** [https://sesssion.onrender.com/](https://sesssion.onrender.com/) – Session ID generator for WhatsApp bots

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🖼️ **Car Gallery** | High‑quality images of tuned BMWs |
| 📝 **Build Stories** | User‑submitted builds with mod lists |
| 🔍 **Search & Filter** | Find cars by model, year, or style |
| 🎨 **Responsive Design** | Works perfectly on mobile & desktop |
| 🔑 **Session ID Tool** | Generate Base64 session IDs for WhatsApp bots |
| 🌙 **Dark Mode** | Easy on the eyes, perfect for night drives |

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, TailwindCSS, JavaScript
- **Backend:** Node.js + Express (for session API)
- **Deployment:** Render.com
- **Version Control:** Git + GitHub

## 🚀 Getting Started (Local Development)

Clone the repository and run the website locally.

```bash
# Clone the repo
git clone https://github.com/Thomas-shelby001/BEAMER-XMD.git
cd BEAMER-XMD

# Install dependencies (if any)
npm install

# Start the development server
npm start
